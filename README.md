# github-user-search-ios
Github OpenAPI로 사용자를 검색하는 iOS 앱입니다


---
## 개발 stack
- Minimum Target : iOS 13
- Swift
- SPM
    + Then
    + Moya
    + SuperEasyLayout
    + RxSwift
    + NSObject-Rx
    + AlamofireImage
- UI : Code-based


---
## 실행 방법
```bash
cd path_to_project_folder/GithubUserSearch
open GithubUserSearch.xcodeproj
```
